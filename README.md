# Snake
Java开发的贪吃蛇小游戏，复习Java基础

# 目的
>寒假又复习了一遍JAVA基础，复习完后用这个小练习巩固一下基础，顺便学习了之前没学awt,虽然现在已经不用，但是我觉得多学一点总是好滴，哈哈啊啊。
>并且这个项目很好的帮助俺理解了内部类的作用

#  功能
1. 键盘上下左右键控制蛇的移动
2. 蛇吃苹果
3. 蛇吃完苹果，分数增加，蛇的身体增加一节。
4. 蛇碰撞到边缘，咬到自己的身体，弹出游戏结束。
5. 按F2重新开始游戏
